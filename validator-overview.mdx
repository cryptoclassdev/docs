### Security Features

Our validator implements multiple layers of security to protect your staked assets:

<AccordionGroup>
  <Accordion icon="shield-check" title="Hardware Security Modules (HSM)">
    Private keys are stored in FIPS 140-2 Level 3 certified hardware security
    modules, ensuring maximum protection against unauthorized access.
  </Accordion>
  <Accordion icon="lock" title="Encrypted Communications">
    All network communications use TLS 1.3 encryption with perfect forward
    secrecy to prevent eavesdropping and man-in-the-middle attacks.
  </Accordion>
  <Accordion icon="eye-slash" title="Air-Gapped Operations">
    Critical operations are performed on air-gapped systems to prevent remote
    attacks and unauthorized access.
  </Accordion>
  <Accordion icon="user-shield" title="Multi-Factor Authentication">
    All administrative access requires multi-factor authentication with hardware
    security keys.
  </Accordion>
</AccordionGroup>
